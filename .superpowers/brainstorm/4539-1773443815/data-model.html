Data Model Design

The knowledge graph at the heart of the encyclopedia — every entity and how they relate

Core Entities

🧘 Master (Person)

  • id — unique slug
  • names — { dharma, birth, honorific } × { en, ja, zh, ko, vi, sa }
  • dates — birth, death, ordination
  • lineage — school, generation number
  • teacher → Master (dharma transmission from)
  • students → Master[] (dharma heirs)
  • temples → Temple[] (founded, resided)
  • biography — rich text, per locale
  • teachings — key teachings, famous koans
  • media — portraits, calligraphy, photos
  • coordinates — birth/death/activity locations

🏯 Temple / Monastery

  • id — unique slug
  • names — per locale
  • location — coordinates, region, country
  • founded — date, founder → Master
  • school — lineage affiliation
  • abbots → Master[] (historical)
  • description — per locale
  • media — photos, maps
  • status — active, historical, destroyed

📜 Teaching / Text

  • id — unique slug
  • title — per locale
  • type — koan, sutra, commentary, poem, talk
  • author → Master
  • collection — e.g., Blue Cliff Record, Mumonkan
  • content — text, per locale
  • era — historical period
  • related → Teaching[], Master[]

🔗 Lineage / School

  • id — unique slug
  • names — per locale (e.g., Rinzai/Linji/임제)
  • tradition — Chan, Zen, Seon, Thiền
  • parent → School (branched from)
  • founder → Master
  • founded — approximate date
  • practices — key methods (koan, shikantaza, etc.)
  • description — per locale
  • active — boolean

Extended Entities (Phase 2+)

🎨 Art / Artifact

Calligraphy, ensō, paintings, sculptures — linked to masters and temples

🧘‍♂️ Practice

Zazen, kinhin, oryoki, sesshin — methods and ceremonies per school

📅 Event

Historical events, councils, persecutions, diaspora — for the timeline

Relationship Graph

  Master ──teaches──▶ Master        (dharma transmission)
  Master ──founded──▶ Temple        (established)
  Master ──wrote────▶ Teaching      (authored)
  Master ──belongs──▶ School        (lineage affiliation)
  School ──branched─▶ School        (historical split)
  Temple ──houses───▶ Art           (collection)
  School ──practices▶ Practice      (methods used)
  Event  ──involves─▶ Master,Temple (historical context)
