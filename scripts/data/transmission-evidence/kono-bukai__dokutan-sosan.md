reducer_notes: |
  Three sources confirm the transmission chain Dokutan Sōsan → Kono Mukai/Bukai:

  1. Terebess Takuju Kosen page: explicit lineage list showing the full chain
     Takuju Kosen → Myoki Soseki → Karyo Zuika → Tankai Gensho →
     Dokutan Sosan → Kono Mukai, Mukai Koryo, Nanshinken (1864-1935).
  2. Terebess Shibayama Zenkei page: biographical lineage lists Dokutan Sōsan
     directly preceding Mukai Koryō (aka Kōno Mukai / Kono Bukai); the same
     page confirms Shibayama received transmission from Kono Roshi and trained
     at Nanzenji under "Abbot Kono Bukai."
  3. Dharma Wheel forum: community-class chart corroborating the same chain.

  Alternate name forms all refer to the same individual (1864-1935):
  Kono Bukai = Kono Mukai = Mukai Koryo = Nanshinken.
